# text_audio_plugin_processor
Aplicación modular de procesamiento de texto y audio en Java, estructurada bajo una arquitectura plugin. Permite agregar nuevos módulos sin modificar el núcleo, integrando funcionalidades como Text-to-Speech, Speech-to-Text y almacenamiento en base de datos.  Tecnologías: Java, Maven, Arquitectura Plugin, TTS/STT, SQL
